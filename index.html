Welcome to NecrOns!

This is my journey in painting a Portal-themed Warhammer 40k Necron Scarab miniature. Completing this project requires both assembly and painting of the miniature.

This site is a beginner's hobby journal for anyone curious about Warhammer 40,000, Portal, or miniature painting and wants to see what getting into the hobby might look like.

The Project at a Glance

I will be documenting a Portal-themed Necron Combat Patrol from start to finish: from cutting pieces off the sprue, to gluing them together, to applying base coats, accents, and final highlights.

  • Assembly: Learn about the tools and steps I use to prepare and assemble the models.
  • Painting: Follow the painting process step-by-step from base coat to final details.

Where to Start

New to miniatures or Warhammer 40,000? Start with the Assembly page to see how I prepare the miniature. When you're ready for color, move on to the Painting page.
